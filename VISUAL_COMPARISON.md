## 🎨 Условное форматирование

### Правило:
```
ЕСЛИ (Оборачиваемость ≤ 14) ТОГДА
  Фон: Красный (rgb(242, 51, 51))
  Текст: Белый + Жирный
ИНАЧЕ
  Обычное форматирование
```

## 📈 Интерпретация значений

### 🔴 КРИТИЧНО (≤14 дней)
```
Значение: 8 дней
Значит: Товар закончится через 8 дней!
Действие: СРОЧНАЯ ЗАКУПКА!
```

### ⚠️ ВНИМАНИЕ (15-30 дней)
```
Значение: 22 дня
Значит: Товар закончится через ~3 недели
Действие: Планировать закупку
```

### ✅ НОРМАЛЬНО (>30 дней)
```
Значение: 214 дней
Значит: Товара хватит на 7 месяцев
Действие: Возможно, избыток товара
```

## 🔢 Примеры расчета

### Пример 1: Критический уровень
```
Остатки: 50 шт
Заказы за 7 дней: 25 шт
────────────────────────────
Заказов в день: 25 ÷ 7 = 3.57 шт/день
Оборачиваемость: 50 ÷ 3.57 = 14 дней 🔴
                                    ↑
                            ГРАНИЦА!
```

### Пример 2: Очень критический
```
Остатки: 30 шт
Заказы за 7 дней: 35 шт
────────────────────────────
Заказов в день: 35 ÷ 7 = 5 шт/день
Оборачиваемость: 30 ÷ 5 = 6 дней 🔴
                                  ↑
                         СРОЧНО ЗАКАЗАТЬ!
```

### Пример 3: Нормальный уровень
```
Остатки: 150 шт
Заказы за 7 дней: 35 шт
────────────────────────────
Заказов в день: 35 ÷ 7 = 5 шт/день
Оборачиваемость: 150 ÷ 5 = 30 дней
                                   ↑
                            НОРМА
```

## 📱 Как это выглядит в реальности

### На компьютере:
- Красный фон: `rgb(242, 51, 51)` - яркий, заметный
- Белый текст: `rgb(255, 255, 255)` - отличный контраст
- Жирный шрифт: дополнительное выделение

### На мобильном:
- Полностью адаптивно
- Красный цвет хорошо виден
- Легко найти критические товары одним взглядом
